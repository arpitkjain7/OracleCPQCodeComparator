Author : Nandhini Vurimella.
Description : Updates the transaction's version number. 
              Sets the Commerce Process 'Version' attribute (version_t) to the value of the CPQ platform-created Version attributes set 'Version' attribute (version_number_versionTransaction_t).  
              This allows all customizations to refer to the Commerce attribute rather than a system attribute.
*****************************************************************************************************/

 return atoi(version_number_versionTransaction_t);
 } else {

}
