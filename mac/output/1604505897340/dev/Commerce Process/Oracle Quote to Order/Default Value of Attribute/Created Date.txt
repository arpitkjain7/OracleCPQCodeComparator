/*********************************************************************
              Function Name: Defaulting the Created Date as MM/dd/yyyy format
              Description: This function is used to Default the Created Date as MM/dd/yyyy format
              User Story #: UID-527
              Date: 26th Feb 2019
              Author : Keerthivasan
*********************************************************************/
tmpdate = strtojavadate(_system_date, "MM/dd/yyyy");
resdate = datetostr(tmpdate,"MM/dd/yyyy");
return resdate;
